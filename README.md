【Mod Introduction】:

TB-LIFE Beautification Mod is a Dol player character beautification mod created by Grace with the assistance of miyako4828 in coding, featuring Version 1 and Version 2.

Version 1 is a 600x300 large canvas module, requiring https://github.com/HCPTangHY/DOL-PaperdollPlus as a prerequisite module. Version 2 is the original size module, with or without Paperdoll+ installation depending on personal preference.

Adapted for game version 0.5.9 and above, currently prioritizing TB1, with TB2 to be drawn as time permits. 【Misalignment is normal, as I haven't drawn it yet.】

If you have any questions about beautification, please post them in the tb-life section of the DOL Chinese Translation Server on Discord or the tb-life Beautification Channel on the Degrees of Lewdity Modding Server. If your native language is not Chinese, please include a machine-translated Chinese version to help me process your request faster.

To prevent rampant piracy, a password has been added, so it can only be imported via modloader and not replaced. Universal mod password: 本免费美化仅发布于dc中英双区，禁止二传商用

【Usage Instructions】:

After downloading the "xx.mod.zip" file, import it into the modloader, enter the password, refresh to enable the BSA, and refresh again to play normally.

【The zip file containing PSDs is for viewing layers or creating derivative works in graphic design software after extraction. Please do not import it into games】
